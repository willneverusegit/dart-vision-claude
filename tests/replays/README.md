# Replay Clips

Dieses Verzeichnis ist fuer kurze deterministische Testclips gedacht, z. B.:
- normaler Treffer
- bounce out
- Hand im Bild
- Dart-Entnahme
- schwierige Beleuchtung

Dateien koennen wegen Groesse ausserhalb von Git verwaltet werden. Wichtig ist:
- Stabile Dateinamen
- Passende Ground-Truth-Sidecars in `tests/ground_truth/`
