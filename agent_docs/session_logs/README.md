# Session Logs

Hier werden nach jeder Agent-Session kurze Protokolle abgelegt.

## Format

Dateiname: `JJJJ-MM-TT_kurzthema.md`

Inhalt:
```
# Session JJJJ-MM-TT: Kurzthema

## Erledigt
- was wurde gemacht

## Probleme
- was schiefging oder unerwartet war

## Gelernt
- Erkenntnisse fuer zukuenftige Sessions

## CLAUDE.md-Anpassungen
- was an CLAUDE.md geaendert wurde (falls zutreffend)
```

## Regeln

- Maximal 15 Zeilen pro Session-Log
- Keine Code-Bloecke, nur Stichpunkte
- Vor neuer Arbeit die letzten 3-5 Logs lesen
